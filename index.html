Number Ping-Pong!

Let's play number ping-pong! Enter a random number and see whether we pop it back to you as a ping or as a pong!

Rules:

We will count up to the number you provide from 1, with the following exceptions:
  1. Numbers divisible by 3 return as "ping"
  2. Numbers divisible by 5 return as "pong"
  3. Numbers divisible by 15 return as "ping-pong"
